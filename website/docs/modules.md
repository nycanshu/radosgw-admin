# Module Guides

- [users](./guides/users): create/get/modify/delete/list/suspend/enable/getStats
- `keys`: generate/revoke
- `subusers`: create/modify/remove
- `buckets`: list/getInfo/delete/transferOwnership/removeOwnership/verifyIndex
- `quota`: user and bucket quota management
- `rateLimit`: user, bucket, and global rate limits
- `usage`: usage reports and trimming
- `info`: cluster information

For full signatures and types, see [API Reference](./api).
